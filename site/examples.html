Run locally

Start any example with pnpm filter

pnpm install
pnpm --filter units-learn-demo dev
pnpm --filter units-todo-demo dev
pnpm --filter units-chat-demo dev
pnpm --filter units-shadcn-gallery dev

Bench fixtures

The benchmark corpus lives beside the examples

In addition to runnable apps, the repository keeps curated .ui and .jsx pairs under bench/cases plus a UI-PS seed library. Those files are part of the practical example surface too, especially for token and transform evaluation.
