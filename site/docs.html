Props and events

Minimal but expressive

key:value
key=@expr
key?=@expr
!click { set(selected:=@item.id) }
on:click={ doThing(@item) }

Runtime

Execution model

  • React renderer with scoped evaluation
  • Custom host renderer API
  • Inline event and expression support
  • Directive flow for if, elif, else, for, slot, and key

Tooling

Authoring and build path

  • Vite plugin for .ui imports
  • Format/lint/token/highlight tools
  • CLI utilities for .ui workflows
  • VS Code language support with snippets, grammar, and icon theme

Agent additions

Compression and neutral IR layers

  • units-agent-middleware for structural UI rewrites
  • units-agent-plugin and units-agent-service for compress-ui delivery
  • units-dom-snapshot, units-ui-ir, units-react-adapter, and units-compiler for extraction and compile flows

Verification

Benchmarks and tests

  • Parser, DSL, system, LLM, React-vs-DSL, and UI-PS benchmark families
  • Node test coverage across runtime, tools, compiler, DOM snapshot, adapters, and plugins
  • Machine-readable reports written under bench/results
