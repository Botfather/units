Minimal DSL for interactive UI

Units is a lightweight UI DSL with fast parsing, small syntax, and both React + custom renderer support.

Read docs See examples

Quick Start

import { parseUnits, renderUnits } from "@botfather/units";

const ast = parseUnits("Card { text 'Hello Units' }");
const node = renderUnits(ast, { user: { name: "Ada" } });
